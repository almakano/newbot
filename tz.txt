Создать веб приложение в котором пользователь:
- сможет добавить новую картинку с описанием,
- просмотреть список всех картинок
- и удалить любую картинку.

Пожелания к технической части:
1. Использовать веб фреймворк Yii2
2. Использовать миграции для создания таблицы в базе данных.
3. Сделать интерфейс с помощью Twitter Bootstrap.
4. Залить проект на github.
